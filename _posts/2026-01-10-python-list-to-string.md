## 문제 설명
문자들이 담긴 배열 `arr`이 주어질 때, 이를 이어 붙인 문자열을 리턴하는 함수를 작성합니다.

## 나의 풀이
가장 효율적인 방법인 `join()` 함수를 사용했습니다.

```python
def solution(arr):
    return "".join(arr)
```

## 💡 이번 문제를 통해 배운 점
오늘 코딩 테스트 연습을 하면서 리스트를 문자열로 합치는 방법을 배웠습니다.

### 1. 문제의 핵심
여러 개의 문자가 담긴 리스트를 하나의 문장으로 만드는 것이 핵심입니다.

### 2. 해결 방법: `join()` 함수
파이썬에서는 `"".join(리스트)`를 사용하면 아주 간단하게 해결할 수 있습니다.
