## 1. 문제 설명
정수 num1, num2가 매개변수로 주어질 때, num1을 num2로 나눈 몫을 return 하도록 solution 함수를 완성해주세요.

### 제한사항
- 0 < num1 ≤ 100
- 0 < num2 ≤ 100

## 1. 나의 풀이
파이썬의 **정수 나눗셈 연산자(`//`)**를 사용하여 간단하게 해결했습니다.

```python
def solution(num1, num2):
    # // 연산자는 나눗셈의 결과에서 소수점 이하를 버리고 몫만 반환합니다.
    answer = num1 // num2
    return answer
```

## 1. 💡 이번 문제를 통해 배운 점
파이썬의 두 가지 나눗셈 연산자
이 문제를 통해 파이썬에서 나눗셈을 처리하는 두 가지 방식을 확실히 정리했습니다.

/ (나눗셈 연산자): 소수점까지 포함한 결과를 반환합니다. (예: 7 / 2 = 3.5)

// (몫 연산자): 나눈 결과의 **정수 부분(몫)**만 반환합니다. (예: 7 // 2 = 3)

사고 과정
처음에는 단순히 나누기(/)를 생각할 수 있지만, 문제에서 요구하는 것이 **'몫'**이라는 점에 주목했습니다. 파이썬은 정수 나눗셈을 위한 전용 연산자를 제공하기 때문에 이를 활용하는 것이 가장 효율적이라는 것을 알게 되었습니다.

## 2. 문제 설명
정수 num1과 num2가 주어질 때, num1과 num2의 합을 return하도록 soltuion 함수를 완성해주세요.
### 제한 사항
- -50,000 ≤ num1 ≤ 50,000
- -50,000 ≤ num2 ≤ 50,000

## 2. 나의 풀이
파이썬의 기본적인 **더하기 연산자(`+`)**를 사용하여 해결했습니다.

```python
def solution(num1, num2):
    return num1 + num2
```

## 2. 💡 이번 문제를 통해 배운 점
기초 연산의 중요성
이 문제는 매우 간단해 보이지만, 프로그래밍의 가장 기초가 되는 산술 연산의 흐름을 이해하는 데 중요합니다.

사고 과정
입력값 확인: 두 개의 정수(num1, num2)가 들어온다는 점을 확인했습니다.

요구사항 파악: 두 수의 '합'을 구해야 하므로 산술 연산자 중 +를 선택했습니다.

제한사항 고려: 파이썬 3는 정수 범위(Integer)를 자동으로 관리해주기 때문에, 문제에서 주어진 범위(-50,000 ~ 50,000)는 별도의 예외 처리 없이 기본 연산으로 충분히 처리 가능함을 인지했습니다.

## 3. 문제 설명
정수 num1과 num2가 주어질 때, num1에서 num2를 뺀 값을 return하도록 soltuion 함수를 완성해주세요.
### 제한 사항
- -50,000 ≤ num1 ≤ 50,000
- -50,000 ≤ num2 ≤ 50,000

## 3. 나의 풀이
파이썬의 기본적인 **빼기 연산자(`-`)**를 사용하여 해결했습니다.

```python
def solution(num1, num2):
    return num1 - num2
```

## 4. 문제 설명
정수 num1, num2가 매개변수 주어집니다. num1과 num2를 곱한 값을 return 하도록 solution 함수를 완성해주세요.
### 제한 사항
- 0 ≤ num1 ≤ 100
- 0 ≤ num2 ≤ 100

## 4. 나의 풀이
파이썬의 기본적인 **곱하기 연산자(`*`)**를 사용하여 해결했습니다.

```python
def solution(num1, num2):
    return num1 * num2
```

파이썬 산술 연산자 정리
+ : 더하기

- : 빼기

* : 곱하기

/ : 나누기

// : 몫

% : 나머지
